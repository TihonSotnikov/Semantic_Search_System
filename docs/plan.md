### План реализации

#### Шаг 1. Скелет системы
*   **Тихон:** Скрипт векторизации и поиска по массиву строк в оперативной памяти.
*   **Андрей:** Запуск FastAPI, создание эндпоинта-заглушки, верстка базовой формы поиска.

#### Шаг 2. Интеграция
*   **Тихон:** Перенос тестовых данных в JSON-файл и обеспечение поиска по ним.
*   **Андрей:** Подключение фронтенда к реальному эндпоинту, вызов функций поиска, рендеринг результатов.

#### Шаг 3. Работа с состоянием (БД)
*   **Тихон:** Внедрение SQLite для хранения текстов и векторов. Реализация функции записи документа.
*   **Андрей:** Создание эндпоинта и формы для загрузки новых документов в базу.
